Course Summary

This course provides an introduction to computational linguistics and natural language processing (NLP). Students will explore how computers can be used to analyze, understand, and generate human language. The course covers fundamental concepts in linguistics as they relate to computational approaches, including phonetics, morphology, syntax, semantics, and pragmatics.

We will examine both traditional rule-based approaches and modern machine learning methods, with particular attention to neural networks and large language models. Students will gain hands-on experience with programming tools and techniques commonly used in computational linguistics research and industry applications.

Learning Outcomes

By the end of this course, students will be able to:

  • Understand fundamental concepts in computational linguistics and their relationship to theoretical linguistics
  • Apply computational methods to analyze linguistic data and solve language-related problems
  • Implement basic NLP algorithms using Python and relevant libraries (NLTK, spaCy, etc.)
  • Evaluate different approaches to computational language analysis, including rule-based and statistical methods
  • Critically assess the capabilities and limitations of current language technologies
  • Design and conduct small-scale computational linguistics experiments
  • Communicate findings effectively through written reports and oral presentations

Prerequisites

Students should have completed an introductory linguistics course (LIN 306 or equivalent) and have basic programming experience in Python or another programming language. No advanced programming knowledge is required, but willingness to learn new technical skills is essential.
